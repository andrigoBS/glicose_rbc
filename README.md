## Project
case-based reasoning to predict glucose level by carbohydrates

## Dataset
https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset

## Requirements
- python >= 3.19

## Run
- donwload dataset
- put dataset on "./in" directory
- run preprocess.py
```sh
$ python ./preprocess.py
```
- run main.py
```sh
$ python ./main.py
```
